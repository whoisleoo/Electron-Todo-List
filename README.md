# Task Hub


##  📅 Sobre o projeto:

 
Task Hub é uma aplicação feita em Electron e React. <br>
cuja o principal intuito é criar uma lista de afazeres.

````
#Instalar dependencias
npm install

# Iniciar back
npm start

# Iniciar front
npm run dev
````
